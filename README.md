# The Keeper's Log (Static Site)

## Preview locally
Unzip and open `index.html` in your browser.

## Host for free (GitHub Pages)
1. Create a GitHub account.
2. Create a new repository (e.g. `keepers-log`).
3. Upload everything in this folder.
4. Repo → Settings → Pages → Deploy from branch → `main` / `(root)`.
5. Your site will be published at a GitHub Pages URL.

## Update content
- Edit `index.html` to update About/Contact and add new Research Log entries.
